# 🎬 Movies Website

> A full-stack **MERN** web application for browsing movies and media with a modern UI and dynamic content.

🔗 **Live Demo:** https://movies-website-eosin.vercel.app/

---

## 📌 Overview

Movies Website is a full-stack application built using the **MERN stack (MongoDB, Express, React, Node.js)**. The goal of this project is to provide users with an intuitive platform where they can explore movies, view details, and optionally save favorites (depending on implemented features). It includes a backend API, frontend React UI, and responsive design.

📦 Getting Started (Local Development)
🧾 Prerequisites

Before running this project, make sure you have:

Node.js installed (https://nodejs.org/
)

MongoDB database (local or hosted)

An API key (if using an external movie API)

🛠️ Installation

Clone the repo

git clone https://github.com/ziad-mohamed-mern/Movies-Website.git
cd Movies-Website

Install dependencies

Backend:

cd backend
npm install

Frontend:

cd ../frontend
npm install

Setup environment variables

Create a .env file in the backend folder and add:

MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
MOVIE_API_KEY=your_api_key  # if applicable
▶️ Running the App
🖥 Backend (API)
cd backend
npm start
🌐 Frontend (React)
cd frontend
npm start

Your frontend will run at http://localhost:3000
 and backend at http://localhost:5000
 by default.
